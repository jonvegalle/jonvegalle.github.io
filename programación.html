Modulo de programacion

La materia de programación trata sobre aprender a darle instrucciones a una computadora para que pueda realizar tareas de manera automática mediante el uso de algoritmos y lenguajes de programación. En esta asignatura se desarrolla el pensamiento lógico para analizar problemas y resolverlos paso a paso, creando soluciones que después pueden convertirse en programas informáticos. También se estudia cómo almacenar información mediante variables, los distintos tipos de datos que existen y la manera en que un programa puede tomar decisiones o repetir acciones usando estructuras de control como condicionales y ciclos. Además, se aprende a trabajar con arreglos o listas de datos y a representar soluciones mediante diagramas de flujo. Todo esto permite desarrollar software, aplicaciones, páginas web o sistemas que ayudan a automatizar procesos y facilitar diversas actividades mediante el uso de la tecnología.
